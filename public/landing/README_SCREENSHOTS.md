# Instructions pour ajouter les captures d'écran

Pour que les captures d'écran s'affichent dans le téléphone de la landing page, veuillez ajouter vos 3 captures d'écran dans ce dossier (`public/landing/`) avec les noms suivants :

## Option 1 : Noms simples (recommandé)
1. `screenshot-1.png` - Capture d'écran 2025-12-05 à 12.51.40
2. `screenshot-2.png` - Capture d'écran 2025-12-05 à 12.52.41  
3. `screenshot-3.png` - Capture d'écran 2025-12-05 à 12.11.25

## Option 2 : Noms avec dates
Si vous préférez garder les noms originaux :
1. `Capture d'écran 2025-12-05 à 12.51.40.png`
2. `Capture d'écran 2025-12-05 à 12.52.41.png`
3. `Capture d'écran 2025-12-05 à 12.11.25.png`

**Format recommandé :**
- Format : PNG ou JPG
- Taille : 320x640px (ratio 9:16 pour mobile)
- Optimisation : Compressées pour le web

Les images seront automatiquement utilisées dans le composant HeroPhone et changeront toutes les 4 secondes avec une belle transition fade.

Une fois les fichiers ajoutés, ils apparaîtront automatiquement dans le téléphone animé de la landing page !
